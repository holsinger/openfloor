Created by: AshDigg
Created on: 2006/12/09
Requires: None
Description
An AJAX contact form.
1. After enabling, create a link somewhere (like in your faq-en.php page). The link is: /module.php?module=ajaxcontact
2. Edit modules/ajaxcontact/ajaxcontact_settings.php with your email address and name.
