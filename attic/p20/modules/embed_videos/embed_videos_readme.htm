Instructions
Modify your HTML tags to allow here. Add this: <youtube><googlevideo><xoinks><break><vimeo><revver><myspace><veoh><wmv><dailymotion><ifilm><metacafe><tubeley><guba>
If you have this error: [Fatal error: TPL: [in ../modules/embed_videos/templates/link_summary_pre_story_content.tpl line 203]: syntax error: embedvideos function does not exist (class.compiler.php, line 473) in] then you need to move the /modules/embed_videos/plugins file(s) into /plugins
